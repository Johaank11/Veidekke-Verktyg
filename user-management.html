Användarhantering i Supabase

Denna guide visar hur du kan lägga till användare manuellt i Supabase för inloggning i verktygshanteringssystemet.

Åtkomst till Supabase Dashboard

  1. Gå till Supabase.com och logga in med ditt konto.
  2. Välj projektet "Johaank11's Project" från projektlistan.
  3. I sidomenyn, klicka på "Authentication" för att hantera användare.

Lägga till en ny användare

  1. I Authentication-sektionen, klicka på fliken "Users".
  2. Klicka på knappen "Add User" eller motsvarande ("+").
  3. Fyll i:
    • Email: användarens e-postadress
    • Password: ett temporärt lösenord
  4. Klicka på "Add User" för att skapa användaren.
  5. Den nya användaren kommer nu att visas i listan och kan logga in på systemet med det angivna lösenordet.

Hantera befintliga användare

  1. I Users-tabellen kan du se alla registrerade användare.
  2. Du kan redigera användarinformation genom att klicka på en användare i listan.
  3. För att inaktivera en användare, välj användaren och klicka på "Delete User" eller använd "Ban User"-funktionen.
Obs! För ökad säkerhet, överväg att aktivera tvåfaktorsautentisering eller e-postvarifikation i Authentication-inställningarna.

Säkerhetsinställningar

Du kan även konfigurera autentiseringsregler i Supabase:

  1. I Authentication-sektionen, klicka på "Settings".
  2. Under "Email Auth", kan du konfigurera:
    • Om e-postbekräftelse krävs
    • Hur lösenordsåterställning ska fungera
    • Minimikrav på lösenord
